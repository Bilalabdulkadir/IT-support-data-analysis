# IT Support Data Analysis

This project analyzes IT support tickets to identify common issues and trends.

## 📊 Tools Used
- Python
- Pandas
- Excel

## 📁 Dataset Example
| Ticket ID | Issue Type | Resolution Time |
|----------|-----------|----------------|
| 001      | Network   | 30 min         |
| 002      | Hardware  | 45 min         |

## 📈 Insights
- Most common issue: Network problems
- Average resolution time: 35 minutes

## 🎯 Goal
Improve IT support efficiency using data-driven insights.
